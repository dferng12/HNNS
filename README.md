# HNNS
HTTP/s Ngrok Nginx server in a Docker image

<strong>SETUP</strong>

To setup the ngrok, we just have to create an account, grab the authtoken and authorize with:

```
./ngrok authtoken <your_token>
```

and that's all!

<strong>RUN</strong>

To launch the server, just type:
```
./launch-server.sh
```
